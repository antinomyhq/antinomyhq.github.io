Welcome to Forge! This quickstart guide will have you up and running in just a few minutes. Forge is an AI-powered development assistant that helps you build, debug, and enhance your projects with intelligent automation.

<Step number={1} title="Install Forge">

Choose your preferred installation method:

**Option A: NPM (Recommended)**
```bash
npm install -g @forge-ai/cli
```

**Option B: Homebrew (macOS)**
```bash
brew install forge-ai
```

**Option C: Direct Download**
Download the latest release from our [GitHub releases page](https://github.com/forge-ai/forge/releases).

**<CheckCircle size={16} style={{verticalAlign: 'middle', marginRight: 6}} /> Success Check:** Run `forge --version` to confirm installation.

</Step>

<Step number={2} title="Verify Installation">

Test that Forge is properly installed and accessible:

```bash
forge --version
```

You should see output similar to:
```
Forge AI Assistant v2.1.0
```

If you encounter any issues:
- **Path not found:** Restart your terminal or add Forge to your PATH
- **Permission denied:** Run with `sudo` on Unix systems or as Administrator on Windows
- **Command not found:** Ensure the installation completed successfully

**<CheckCircle size={16} style={{verticalAlign: 'middle', marginRight: 6}} /> Success Check:** Version number displays correctly.

</Step>

<Step number={3} title="Initialize Your First Project">

Navigate to your project directory and initialize Forge:

```bash
cd your-project-directory
forge init
```

This command will:
- Create a `.forge` configuration directory
- Set up default rules and preferences
- Scan your project structure
- Generate initial context understanding

**Example output:**
```
âœ“ Analyzing project structure...
âœ“ Creating .forge configuration...
âœ“ Setting up default rules...
âœ“ Project initialized successfully!
```

**<CheckCircle size={16} style={{verticalAlign: 'middle', marginRight: 6}} /> Success Check:** `.forge` directory created in your project root.

</Step>

<Step number={4} title="Configure Your Preferences">

Set up your development preferences:

```bash
forge config
```

Key settings to configure:
- **Editor:** Your preferred code editor (VS Code, Vim, etc.)
- **Language:** Primary programming language for the project
- **Framework:** Web framework if applicable (React, Vue, etc.)
- **Testing:** Preferred testing framework

**Interactive setup example:**
```
? What's your preferred editor? VS Code
? Primary language for this project? TypeScript
? Web framework? React
? Testing framework? Jest
```

**<CheckCircle size={16} style={{verticalAlign: 'middle', marginRight: 6}} /> Success Check:** Configuration saved to `.forge/config.json`.

</Step>

<Step number={5} title="Start Your First Session">

Launch Forge in interactive mode:

```bash
forge
```

You'll see the Forge prompt:
```
ðŸ”¨ Forge AI Assistant
Type /help for commands, /exit to quit

forge> 
```

Try your first command:
```
forge> help me understand this codebase
```

Forge will analyze your project and provide insights about:
- Project structure and architecture
- Key files and dependencies
- Potential improvements
- Development workflow suggestions

**<CheckCircle size={16} style={{verticalAlign: 'middle', marginRight: 6}} /> Success Check:** Interactive session starts and responds to commands.

</Step>

<Step number={6} title="Try Advanced Features">

Explore Forge's powerful capabilities:

**Git Integration:**
```bash
forge --git-diff
```
Analyzes your current changes and suggests improvements.

**Autonomous Mode:**
```bash
forge --agent muse
```
Runs Forge in autonomous mode for complex tasks.

**Custom Rules:**
```bash
forge --rules custom-rules.md
```
Apply project-specific guidelines and constraints.

**File Watching:**
```bash
forge --watch
```
Monitors file changes and provides real-time assistance.

**<AlertCircle size={16} style={{verticalAlign: 'middle', marginRight: 6}} /> Note:** Some features require additional setup or API keys.

**<CheckCircle size={16} style={{verticalAlign: 'middle', marginRight: 6}} /> Success Check:** You can run different Forge modes and see their outputs.

</Step>

<Step number={7} title="Essential Commands Quick Reference" last={true}>

Master these key commands for efficient workflows:

| Command | Purpose | Example |
|---------|---------|---------|
| `forge` | Start interactive session | `forge` |
| `forge --agent muse` | Start in autonomous mode | `forge --agent muse` |
| `forge --git-diff` | Focus on current changes | `forge --git-diff` |
| `forge --help` | Show all options | `forge --help` |
| `/exit` | End current session | `/exit` |
| `/reset` | Clear conversation history | `/reset` |
| `/rules` | Show custom rules | `/rules` |
