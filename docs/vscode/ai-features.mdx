This documentation is being updated with new features. Check back soon!

The Forge VS Code extension provides AI-powered features that integrate directly into your editor workflow.

## Features

- **Delegate to Forge (CodeLens)** - Click to delegate TODO/FIXME comments to Forge
- **Fix using Forge (Quick Fix)** - Use Forge to fix code issues from the lightbulb menu
- **Background Tasks** - Run multiple Forge tasks in parallel
