# MCP Integration

:::note Coming Soon
This documentation is being updated with new features. Check back soon!
:::

The Forge VS Code extension includes an MCP (Model Context Protocol) server that exposes VS Code's Language Server Protocol features to Forge CLI.

## Features

- **LSP Tools** - Access hover, go-to-definition, find-references, and more from Forge
- **Symbol Operations** - Rename symbols, get diagnostics, and explore document symbols
- **Workspace Integration** - Search symbols across your entire workspace
