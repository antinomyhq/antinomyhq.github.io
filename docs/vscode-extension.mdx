# VS Code Extension

Copy file references directly from VS Code to use with Forge. Select code and press `Ctrl+U` to copy the reference in Forge's format.

![Forge VS Code Extension Demo](../static/blog/demo_vscode.gif)

## <Download size={20} style={{verticalAlign: 'baseline', marginRight: 8}} /> Installation

### Prerequisites

- **VS Code**: Version 1.102.0 or higher
- **Forge**: Must be installed and available in your system PATH
- **Node.js**: Version 16+ (required for Forge)

### Install from VS Code Marketplace

1. Open VS Code
2. Go to the Extensions view (`Ctrl+Shift+X` or `Cmd+Shift+X`)
3. Search for "Forge Code"
4. Click **Install** on the official extension by ForgeCode

**Direct Installation Link**: <CustomLink href="https://marketplace.visualstudio.com/items?itemName=ForgeCode.forge-vscode">VS Code Marketplace</CustomLink>

### Alternative Installation Methods

**Command Line Installation:**
```bash
code --install-extension ForgeCode.forge-vscode
```

> **Note:** Works on Windows, macOS, and Linux with VS Code's built-in CLI.

## <Zap size={20} style={{verticalAlign: 'baseline', marginRight: 8}} /> Features

### <Copy size={16} style={{verticalAlign: 'baseline', marginRight: 6}} /> File Reference Copying

Automatically formats file references in Forge's syntax.

**Format:**
```
@[<filepath>:<line start>:<line end>]
```

### How it works

**Selection behavior:**
- **No selection or entire file**: `@[src/components/Button.tsx]`
- **Single line**: `@[src/components/Button.tsx:42:42]`
- **Multiple lines**: `@[src/utils/helpers.js:15:28]`

**Usage examples:**
```
> Help me debug this function @[src/auth/login.js:45:67]
> Review this component @[components/UserProfile.tsx:12:89]
> Refactor this utility @[utils/dataProcessor.js:23:45]
```

### <Keyboard size={16} style={{verticalAlign: 'baseline', marginRight: 6}} /> Shortcuts

- **Copy File Reference**: `Ctrl+U`
- **Start New Forge Session**: Available in Command Palette
- **Command Palette**: "Copy File Reference"
- **Context Menu**: Right-click → "Copy Forge Reference"

## Tips

- Select the specific code you want help with instead of huge files
- Include some context around the problem area
- You can paste multiple file references in one Forge conversation
- Works with any file, any project

## Troubleshooting

### Common issues

**Extension not working?**
- Make sure Forge is installed: `forge --version`
- Restart VS Code after installing the extension
- Check VS Code version (requires 1.102.0+)

**Keyboard shortcut conflicts?**
- Go to File → Preferences → Keyboard Shortcuts
- Search for "Forge" to customize the shortcut

### Getting help

**If you're still having trouble:**
1. Check View → Output → Select "Forge" for error logs
2. Report bugs on <CustomLink href="https://github.com/antinomyhq/forge/issues">GitHub</CustomLink>
3. Ask in our community discussions
