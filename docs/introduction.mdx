# Forge Code overview

Forge Code is an AI software engineering agent that runs in your terminal. It understands your codebase and helps you
write code using natural language instructions.

## Basic usage

**Get started in under 2 minutes:**

```bash
cd path/to/your/project/directory
npx forgecode@latest
```

For detailed installation instructions, see <CustomLink href="/docs/installation">Setup Forge</CustomLink>.
