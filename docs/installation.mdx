# Installation & Setup

Get Forge running on your system in under 5 minutes. This guide covers installation, authentication setup, and project initialization.

## Install and authenticate

<Steps>
  <Step title="Install Forge">
    Install <CustomLink href="https://nodejs.org/en/download">Node.js 16+</CustomLink>, then run:

    ```bash
    npm install -g forgecode@latest
    ```

:::caution
    Do <strong>not</strong> use <code>sudo npm install -g</code> as this can lead to permission issues and security
    risks. If you encounter permission errors, see our{" "}
    <CustomLink href="/docs/npm-troubleshooting">Node.js & npm Troubleshooting</CustomLink>
    guide for safe solutions including npm directory configuration and Node.js version managers like nvm.
:::

  </Step>
  
  <Step title="Navigate to your project">
    ```bash
    cd your-project-directory
    ```
  </Step>
  
  <Step title="Start Forge">
    ```bash
    forge
    ```
  </Step>
  
  <Step title="Complete authentication">

    1. **AI Providers**: Configure Forge to use <CustomLink href="/docs/custom-providers">OpenAI, Anthropic, or other providers</CustomLink> with your own API keys for custom setups.
  </Step>
</Steps>

## Initialize your project

After installing Forge, you can configure additional AI providers beyond the default. Follow the <CustomLink href="../custom-providers">Provider Configuration</CustomLink> guide to set up alternative providers like OpenAI, Anthropic, or OpenRouter.
